## Configurations

### config.yml
```yaml
# Whether to replace the overworld generator
overworld: true
# Whether to replace the nether generator
nether: true
```

## Compiling
1. Install [Maven](https://maven.apache.org/).
2. Run `mvn clean package`. The compiled JAR can be found in the `target/` directory.
